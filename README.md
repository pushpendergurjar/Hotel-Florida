# RESTRO CAFE FLORIDA

Premium, modern, and visually stunning restaurant website template built with HTML, CSS, and Vanilla JS.

## Overview
This repository contains the complete frontend implementation for RESTRO CAFE FLORIDA. It features a dark luxury aesthetic, crafted explicitly to convert visitors into reservations.

## Features
- **Dark Luxury Aesthetic:** Custom color palette featuring Deep Black (`#0D0D0D`), Gold (`#C8A96A`), Soft Cream (`#F5F2EC`), and Burgundy (`#6A1E2D`).
- **Fully Responsive:** Mobile-first layout ensuring perfect display on all devices from smartphones to large desktop monitors.
- **Scroll Animations:** Smooth fade and slide animations triggered contextually on scroll using IntersectionObserver.
- **Interactive Menu:** Tabbed menu switching without page reloads.
- **High-Converting Call to Actions:** Strategically placed reservations buttons and trust markers (4.8/5 Review Badge).
- **Premium Interactivity:** Hover zoom effects, glassmorphic navigations, glowing CSS highlights.

## Getting Started
It's a static repository. 
1. Clone the repository.
2. Open `index.html` in your favorite browser.
3. No build tools required!

## Hosting on GitHub Pages
1. Go to your repository settings on GitHub.
2. Navigate to "Pages" on the left sidebar.
3. Under "Build and deployment", select the "main" branch.
4. Click Save. Your website will be live in a few minutes.
